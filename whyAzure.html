Pourquoi héberger un site web statique sur Azure ?


  • L'hébergement de notre contenu dans Azure Storage nous permet d'utiliser des architectures sans
    serveur qui incluent des fonctions Azure et d'autres services en tant que les services de la platforme (PaaS).

  • Azure Storage static website hosting C'est une excellente option dans les cas où nous n'avons pas besoin
    d'un serveur web pour afficher le contenu.

  • Les utilisateurs peuvent afficher le contenu du site à partir d'un navigateur en utilisant l'URL publique du site web. Nous
    pouvons trouver l'URL en utilisant le portail Azure.
