Les étapes suivies pour effectuer le déploiement d'un site web sur Azure


Pour déployer un site web statique sur Azure, vous avez besoin :


  • Un abonnement Azure actif

  • Nous avons besoin des fichiers et dossiers de notre site web

  • Nous avons besoin d'installer Microsoft Azure Storage Explorer sur notre machine

  • étape1 Création d'un groupe de ressources

    • J'ai créé un groupe de ressources en allant dans le portail Azure et en cliquant sur "Créer une ressource" dans

    le coin supérieur gauche. J'ai sélectionné "Groupe de ressources" dans la liste des catégories. Ensuite, j'ai donné

    un nom au groupe de ressources et j'ai choisi la région où je voulais le créer.


  • étape2 création de storage account

    • dans le groupe de ressources que j'ai créé précédemment J'ai sélectionné (Storage account) pour créer un compte de stockage,

    j'ai rempli les détails du formulaire, tels que le nom du compte de stockage, l'emplacement, le niveau de performance,


    voici un example




    j'ai ensuite configuré les paramètres avancés et réseau en sélectionnant les options appropriées,Pour les paramètres avancés,

    j'ai configuré des options telles que l'accès public, la configuration IP et les règles de pare-feu, après la configuration j'ai cliqué sur creat pour

    créer le compte de stockage,Une fois que le compte de stockage est créé, nous pouvons l'utiliser pour héberger le site web.


  • étape3

    • Une fois que j'ai créé mon compte de stockage, j'ai accédé au menu de gestion de données dans le portail Azure et j'ai sélectionné l'option

    "Web statique" dans le menu de gauche,Cette option m'a permis de configurer mon compte de stockage pour héberger un site web statique.


    voici un example





  • étape4

    • J'ai téléchargé mon site web statique en utilisant Microsoft Azure Storage Explorer. Après m'être connecté à mon compte de stockage Azure, j'ai

    accédé au conteneur de stockage (blob container) créé pour héberger mon site web. Ensuite, j'ai cliqué sur le bouton "Uploader" dans la barre

    d'outils et j'ai sélectionné les fichiers de mon site web à télécharger. Une fois le téléchargement terminé, j'ai vérifié que mon site web

    était en ligne en accédant à l'URL de point de terminaison fournie dans le menu "Web statique" de mon compte de stockage.
